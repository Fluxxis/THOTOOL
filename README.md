## Важно
Сайты могут отдавать 200/редиректы для пустых страниц или блокировать автоматические запросы. Поэтому “найдено” — это эвристика.
## ВАЖНО про 404 (почему у тебя сейчас 404)
Если `/` отдаёт 404, это значит что Vercel *не видит* `index.html` в output/root.
В этой версии я положил `index.html`, `sites.json`, `loading.gif` и в корень, и в `public/` — чтобы проект переживал неправильные настройки Output Directory.

### Настройки Vercel (должно быть так)
- Framework: Other
- Build Command: пусто
- Output Directory: пусто (или `public` тоже ок для этого архива)
- Root Directory: ./ (корень репо)

### Быстрый тест после деплоя
Открой:
- `/` (главная)
- `/index.html`
- `/api/sites`
- `/loading.gif`
